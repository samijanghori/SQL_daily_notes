SQL join

is used to combine rows form two or more tables based on a related column between them

4 Types of join

  • (inner) join
  • (outer) left join
  • (outer) right join
  • (outer) full join

inner join

select records that have match in both tables

is used by the following code :

SELECT ProductID, ProductName, CategoryName FROM Products INNER JOIN Categories ON Products.CategoryID = Categories.CategoryID;
